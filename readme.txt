Quadratic Equation Calculator is a free tool that provides a detailed solution to quadratic equations by supplying the values of a, b and c.

[https://www.calculator.io/quadratic-equation-calculator/](https://www.calculator.io/quadratic-equation-calculator/)

== Usage ==

Add the Quadratic Equation Calculator shortcode to your page, post or sidebar:

`[ci_quadratic_equation_calculator]`

Add the following code to your template where you would like the Quadratic Equation Calculator to appear:

`<?php display_ci_quadratic_equation_calculator(); ?>`

== Installation ==

1. Upload the Quadratic Equation Calculator /ci_quadratic_equation_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Quadratic Equation Calculator plugin through the "Plugins" menu in WordPress.
